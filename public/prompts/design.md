# Category: Design

## UI Component Brainstorm
**Description**: Generates component design ideas and variations for specific use cases

```
I need to design a UI component for:
[DESCRIBE USE CASE]

Requirements:
- [REQUIREMENT 1]
- [REQUIREMENT 2]
- [REQUIREMENT 3]

Please provide:
1. 3 different design approaches
2. Pros and cons of each
3. Best practices for this component
4. Accessibility considerations
5. Mobile responsiveness tips

Be specific with visual descriptions.
```

## Color Palette Designer
**Description**: Creates harmonious color palettes with accessibility considerations

```
Create a color palette for:
[DESCRIBE PROJECT/BRAND]

Requirements:
- Mood/vibe: [DESCRIBE]
- Primary color: [COLOR]
- Use cases: [LIST]

Provide:
1. Primary, secondary, and accent colors
2. Neutral palette (grays/whites/blacks)
3. Accessibility contrast ratios
4. Suggested color combinations
5. Use cases for each color
6. Hex/RGB values

Consider WCAG AA standards.
```

## Layout System Designer
**Description**: Creates comprehensive grid and spacing systems for design consistency

```
Design a layout system for:
[PROJECT DESCRIPTION]

Consider:
- Target screen sizes: [LIST]
- Content types: [LIST]
- Design principles: [LIST]

Provide:
1. Grid structure (columns, gutters)
2. Spacing scale (margins, padding)
3. Typography scale
4. Component sizing recommendations
5. Responsive breakpoints
6. Naming conventions
```

