# Category: Writing

## Content Brief Generator
**Description**: Creates detailed content briefs with objectives, audience, and key messaging

```
Create a content brief for:
[CONTENT TOPIC]

Target audience: [AUDIENCE]
Format: [BLOG POST / ARTICLE / SOCIAL / OTHER]
Goal: [MARKETING / EDUCATIONAL / ENTERTAINING / OTHER]

Include:
1. Content objective and key message
2. Target audience analysis
3. Tone and voice guidelines
4. Key points to cover
5. Structure and outline
6. Call-to-action recommendations
7. SEO considerations
8. Word count recommendations
```

## Blog Post Outline Creator
**Description**: Creates comprehensive, SEO-optimized blog post outlines

```
Create a detailed outline for a blog post about:
[TOPIC]

Target audience: [AUDIENCE]
Key objective: [GOAL]

Structure as:
1. Compelling headline and subheading
2. Introduction (hook and preview)
3. Main sections (3-5 with sub-points)
4. Key takeaways
5. Call-to-action
6. Include keyword suggestions for each section

Make it actionable and engaging.
```

## Email Newsletter Template
**Description**: Generates effective email newsletter templates with proven structure

```
Create an email newsletter template for:
[COMPANY/TOPIC]

Newsletter focuses on: [TOPICS]
Target frequency: [WEEKLY/MONTHLY]

Include:
1. Subject line ideas (3 variations)
2. Welcome/introduction section
3. Main content sections (structure)
4. Call-to-action placement
5. Footer content
6. Personalization suggestions
7. Best practices for engagement

Make it professional and readable.
```

## Social Media Caption Writer
**Description**: Writes engaging social media captions optimized for platform and audience

```
Write engaging social media captions for:

Platform: [PLATFORM - LINKEDIN/TWITTER/INSTAGRAM/TIK TOK]
Post type: [IMAGE/VIDEO/CAROUSEL]
Topic: [TOPIC]
Vibe: [TONE]

Provide:
1. Main caption (platform-optimized length)
2. 2 alternative versions
3. Relevant hashtags (if applicable)
4. Call-to-action suggestions
5. Best time to post
6. Engagement hooks

Match the platform's best practices.
```
