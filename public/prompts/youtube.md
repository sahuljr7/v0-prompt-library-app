# Category: YouTube

## Video Title Generator
**Description**: Creates catchy, SEO-optimized YouTube video titles with hooks and keywords

```
I need help creating an engaging YouTube video title. Here's my video concept:

[VIDEO CONCEPT]

Please provide 5 alternative titles that:
1. Include relevant keywords
2. Create curiosity or urgency
3. Are under 60 characters
4. Follow YouTube best practices
5. Include a compelling hook

Return as a numbered list.
```

## Video Description Writer
**Description**: Writes comprehensive YouTube video descriptions optimized for engagement and discovery

```
Write a YouTube video description for a video about:
[TOPIC]

The description should:
1. Start with a compelling hook in the first 2 lines
2. Include the main topic and key points
3. Have clear sections (Timestamps, Links, etc.)
4. Include relevant hashtags (3-5)
5. End with a call-to-action
6. Be optimized for searchability

Format the output with markdown for readability.
```

## Thumbnail Concept Developer
**Description**: Generates creative thumbnail concepts with design tips for maximum click-through rate

```
Create 3 unique thumbnail concepts for a YouTube video about:
[TOPIC]

For each concept, provide:
1. Visual description
2. Color scheme
3. Text overlay (if any)
4. Design elements
5. Why this would work for CTR

Keep concepts high-contrast and clear for small screens.
```

## Video Script Outliner
**Description**: Creates structured video script outlines with timing suggestions and engagement points

```
Create a detailed video script outline for:
[TOPIC]
Target duration: [MINUTES] minutes

Include:
1. Hook (0-5 seconds) - Compelling opening
2. Introduction (5-15 seconds)
3. Main content (sections with sub-points)
4. Examples or demonstrations
5. Conclusion with CTA
6. Timing estimates for each section

Keep language conversational and engaging.
```
